# 🍳 黑暗料理模拟器 — One Page Design

---

## 🎯 一句话描述

> **用随机抽到的烹饪手法，把采购来的食材加工成一道"黑暗料理"——越离谱越好笑，然后分享给朋友。**

---

## 🧩 核心体验

| 关键词 | 说明 |
|--------|------|
| **惊喜 > 优化** | 玩家追求的不是"最优解"，而是"天哪，这是什么东西" |
| **搞笑 > 完美** | 切牛奶、炸冰糖、三重裹粉——荒诞组合就是最大的乐趣 |
| **分享 > 通关** | 一局生成一张截图，菜名+结局+评语天然适合社交传播 |

---

## 🕹️ 核心玩法循环

```
┌─────────────┐     ┌──────────────────────┐     ┌──────────────┐
│  🛒 超市采购  │ ──→ │  🍳 厨房烹饪（3回合）  │ ──→ │  📊 结算&分享  │
│  一键随机选食材 │     │  买手法卡 → 排卡 → 执行 │     │  菜名/结局/评语 │
└─────────────┘     └──────────────────────┘     └──────────────┘
        ↑                                                 │
        └─────────────── 🔄 再来一局 ←───────────────────┘
```

**一局流程（约 2-3 分钟）：**

1. **采购**：一键随机生成食材篮（10格空间，5种采购风格），可微调后确认
2. **烹饪**（×3回合）：每回合 → 商店买手法卡 → 操作台排卡 → 执行烹饪
3. **结算**：生成菜品名、达成结局、解锁标签，一键分享截图

---

## 📦 内容规模

| 维度 | 数量 | 说明 |
|------|------|------|
| 食材 | 24种 | 8大类 × 3种（肉/蛋奶/主食/糖/蔬菜/水果/油脂/调味料） |
| 烹饪手法 | 14种 | 4大流派：🔥加热(4) / 🔪处理(4) / 🎨调味(3) / 🔗组合(3) |
| 主结局 | 28种 | 7档卡路里 × 4档品质，如"反物质残渣""满汉全席""卡路里黑洞" |
| 标签结局 | 12种 | 暴击触发特殊标签，如"烟熏""刀工""艺术品""入味" |
| 稀有组合 | 8种 | 特定食材+手法组合，如"岩浆蛋糕🌋""分子料理🧊""完美平衡👼" |
| **总收集** | **62种** | 结局图鉴系统，驱动复玩 |

---

## ⚙️ 关键系统

### 资源转换流水线

```
食材属性（卡路里/色/香/味）
        ↓ 叠加
手法卡效果（按执行顺序逐张结算）
        ↓ 15%概率暴击（效果翻倍+搞笑特效）
四维最终数值
        ↓ 7档评级
结局 + 菜品名 + 评语
```

### 经济系统（极简）

| 资源 | 用途 | 获取 |
|------|------|------|
| 🧺 购物篮（10格） | 采购食材 | 局外一次性 |
| 🔥 厨房经费（8/10/12=30） | 购买手法卡 | 每回合递增发放 |

### 操作台递增

| 回合 | 名称 | 槽位数 |
|------|------|--------|
| R1 | 预处理阶段 | 1 |
| R2 | 烹调阶段 | 2 |
| R3 | 收尾阶段 | 3 |

---

## 🎨 菜品命名规则

```
[手法修饰词][食材词][菜式后缀]
```

- 同种手法全局计数合并：双重/三重/四重/疯狂
- 无分隔符直接连接，按手法默认顺序排列
- 示例：`双重煎烤牛肉蘑菇炸物`、`疯狂切调味鸡蛋拼盘`

---

## 💡 为什么好玩？

| 设计支柱 | 实现方式 |
|----------|----------|
| **低门槛** | 一键采购、3回合、每回合只需"买卡→放卡→开火"，2分钟一局 |
| **高惊喜** | 14种手法 × 24种食材 × 随机暴击 = 海量荒诞组合，每局不同 |
| **强传播** | 搞笑菜名+结局+评语 → 自动生成分享图（带二维码） |
| **有粘性** | 62种收集元素 + NEW标记 + 结局图鉴进度条，驱动"再来一局" |

---

## 🎵 氛围

- **视觉**：暖色系（amber/orange渐变）、emoji 驱动、移动端优先
- **音频**：轻松 BGM + 7种卡通音效（购买/放置/烹饪/暴击/结算/分享/点击）
- **角色**：🧑‍🍳 大厨实时评价（吐槽系），全程陪伴烹饪过程
- **调性**：轻松搞笑，不较真，不内卷
